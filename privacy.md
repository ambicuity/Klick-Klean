# Klick Klean Privacy Policy

Last updated: May 1, 2026

## Overview

Klick Klean is a browser extension that helps you delete selected browsing data from Chrome. This privacy policy explains what data we collect, how we use it, and your rights.

## Data We Do Not Collect

Klick Klean operates entirely within your browser. We do **not** collect, transmit, or store any of the following on external servers:

- Browsing history
- Cookies or site data
- Download history
- Form data or autofill entries
- Cache or storage data
- Personal identifiers
- Usage analytics or telemetry

## Data Stored Locally

The only data stored by Klick Klean is kept locally on your device using Chrome's extension storage API:

- **Your settings**: selected cleanup categories, default scope, time range, and preset preferences
- **Allowlist**: sites you choose to exclude from global cleanup (stored as origin URLs)

This data never leaves your device. If you uninstall the extension, this data is automatically removed.

## Permissions

Klick Klean requests the following Chrome permissions:

- **browsingData**: Required to delete the browsing data types you select
- **notifications**: Optional, used only to show cleanup progress notifications (can be disabled in settings)
- **storage**: Required to save your settings and allowlist locally
- **tabs**: Required to detect the current site origin for current-site cleanup scope

## Third Parties

Klick Klean does not integrate with any third-party services, analytics platforms, advertising networks, or tracking pixels. There are no external API calls.

## Open Source

Klick Klean is open source. You can inspect the full source code to verify these privacy claims.
